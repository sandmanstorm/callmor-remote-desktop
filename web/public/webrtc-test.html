Callmor WebRTC P2P Test

Establish a peer-to-peer data channel through the relay server

How to test:
  1. Open this page in two separate browser tabs
  2. In Tab 1: select Agent role, click Connect to Relay
  3. In Tab 2: select Browser role (same Machine ID), click Connect to Relay
  4. In Tab 2 (Browser): click Start WebRTC Call — this sends the offer
  5. Wait for "P2P Connected" status — then type messages in the chat box
Messages in the chat go directly P2P (not through the relay).
